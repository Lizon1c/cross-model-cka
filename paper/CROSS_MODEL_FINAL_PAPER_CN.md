# EVO2 与 RFdiffusion3 的跨模型 CKA 对齐反映低频平滑偏差而非结构收敛

## EVO2 (DNA 语言模型, 7B) ↔ RFdiffusion3 (结构扩散模型) 在 AsCas12f1 上的研究

---

### 摘要

我们测试了 DNA 语言模型 (EVO2) 与蛋白质结构扩散模型 (RFdiffusion3) 是否在 AsCas12f1 CRISPR-Cas 位点上产生收敛于相同物理残基的注意力模式。将蛋白质长度修正为 422 aa (UniProt A0A2U3D0N8)，并通过密码子池化 (3 nt → 1 aa 均值池化，一种降维步骤) 对齐坐标后，我们扫描了全部 160 个 EVO2 MHA 头与 1,216 个 RF3 token 级头 (共 194,560 对)。

最强原始头对达到全长 CKA = 0.851。图 1B 展示了 max-null 分布：null 最大 CKA 分布均值为 0.099，范围为 [0.049, 0.522]，99.9 百分位为 0.512。全扫描 max-null 置换检验 (2,000 次置换, p ≈ 0.0005) 证实该对齐不能用随机残基顺序打乱解释。

然而，大量控制实验表明，原始 CKA 主要受共享的低频坐标几何主导，而非结构域特异的功能对齐。关键发现：

1. **密码子池化不反映阅读框解码。** 移码鲁棒性来自局部核苷酸特征相关性加上 3-mer 均值池化，而非密码子边界特异性。

2. **位置基线在分段尺度上匹配或超过真实 EVO2。** 在 60–120 aa 滑动窗口上，过剩 CKA (真实 EVO2 减位置基线) 强负 (−0.43 至 −0.48)，且不到 1.1% 的窗口显示过剩 > 0.05。

3. **结构域富集在位置感知控制下不成立。** TNB 和 REC 在朴素残基置换零假设下的表观富集被连续区间和循环移位零假设消除。在 TNB 结构域内，位置基线 (CKA = 0.672) 大幅超过真实 EVO2 (CKA = 0.412; 过剩 = −0.260)。

4. **高 CKA 是头特异的且位置敏感的。** 仅 18% 的 RF3 头对纯位置编码产生强响应 (CKA > 0.5)。最佳原始对的 RF3 头 (mod0076_h00) 正弦位置 CKA = 0.595，确认为位置敏感头。

5. **同源二聚体链不对称性可忽略。** Chain A 与 Chain B 的 CKA 值在所有对中相关度 r = 0.975。

6. **突变诱导的 delta 对齐不存在突变特异性。** 300 个突变体的 delta CKA (0.207) 与突变对打乱零假设 (0.212, p = 0.42) 不可区分。去掉共同 delta 均值后 CKA 崩塌至 0.041。突变位点局部窗口和 3D 接触邻域的 delta CKA 均为零。检索分析随机 (top-1 1%, 平均排名 102)。

**结论：** EVO2–RF3 的原始全长 CKA 是真实、非随机且头特异的，但它反映的是两个架构深层位置混合诱导的通用低频平滑偏差，而非结构域级生物内容对齐。在分段和结构域尺度上，显式位置基函数系统性超过真实 EVO2 特征。朴素零假设下的表观结构域富集被位置感知控制消除，且不存在位置弱头对的高 CKA。静态和扰动分析均不支持 EVO2 捕获超越坐标几何的结构域特异功能信息的解释。

**数据和代码：** `scripts/` 和 `output_heads/evo2_full10/`

## 1. 方法

### 1.1 坐标和对齐

**蛋白质：** 每条链 422 aa (UniProt A0A2U3D0N8, 全长 AsCas12f1)。

**RF3 token：** 1,145 = 3 特殊 + 422 (链 A) + 422 (链 B) + 222 (链 C, sgRNA) + 38 (链 D, 靶向 DNA) + 38 (链 E, 非靶向 DNA)。蛋白质分析除特别说明外使用 RF3 链 A。

**EVO2 位点：** 1,857 bp = 168 bp (天然上游侧翼, NCBI NZ_MPDK01000047.1) + 1,269 bp CDS (422 aa × 3 + TAA 终止) + 172 bp (tracrRNA 基因) + 78 bp (天然下游侧翼) + 78 bp (CRISPR 阵列: repeat–spacer–repeat) + 92 bp (PAM + 靶向上下文)。密码子表来自 Sulfoacidibacillus thermotolerans 基因组 (69 CDS, 20,028 密码子, GC 38%)。

**密码子池化：** 3 nt → 1 aa 均值池化将 1,857-bp EVO2 位点映射到 422 残基级坐标。这是降维步骤，而非阅读框解码声明。

### 1.2 度量

- **linear_cka:** 中心化 Gram 矩阵 CKA，范围 [0, 1]
- **Per-residue score:** 距离剖面的严格 Pearson 相关，保证 [−1, 1]
- **Segment CKA:** 残基连续窗口上的 CKA
- **Excess CKA:** 真实 EVO2 CKA 减去位置基线 CKA
- **Max-null:** 每次置换中，所有 160 个 EVO2 头残基坐标同步打乱 (RF3 固定)，重算完整 160 × 1,216 CKA 矩阵并记录最大值

### 1.3 位置基线

- **正弦位置编码：** 128 维 RoPE 风格编码
- **多项式位置趋势：** 残基特征对多项式基 {1, t, t², t³} 的回归

### 1.4 头选择

160 个 EVO2 MHA 头 × 1,216 个 RF3 token 级头，在 GPU 上预计算完整 CKA 矩阵。

## 3. 讨论

### 3.1 全局 CKA 反映坐标几何而非功能对齐

原始 CKA = 0.851 是真实、非随机、头特异的对齐信号。它经受全扫描 max-null 校正 (p ≈ 0.0005) 和残基顺序打乱。然而，它主要由沿蛋白链的共享低频坐标几何驱动，而非结构域特异功能内容。

### 3.2 为何平滑性主导：机制的而非语义的

高 CKA 不需要 EVO2 计算 RF3 式的链拓扑或结构域特异结构。系统合成信号测试表明，对齐由沿残基轴的共享低频自相关驱动。机制是机制性的：两个架构在深层进行大量位置混合——RF3 的 pair-bias 在 50 个扩散 block 中传播使 token 特征沿链轴平滑，EVO2 的 RoPE 注意在 5 个深层中累积积分序列-位置信息。CKA 作为 Gram 矩阵相似性度量，强烈奖励共享低频结构，无论其生物起源。

### 3.3 局限性与未来工作

- **突变面板：** 10 个随机突变体跨突变分析显示高度稳定性，支持坐标几何解释
- **密码子表：** 使用天然 S. thermotolerans 密码子表，移码不变性表明影响有限
- **Max-null 范围：** 2,000 次置换建立 p ≈ 0.0005

### 3.4 Null 层次总览

| Null / 控制 | 检测内容 | 结果 |
|---|---|---|
| 残基打乱 | 残基顺序依赖性 | CKA 崩塌 (0.85 → 0.01): 非随机顺序 |
| Max-null 全扫描 | 多重检验 (194,560 对) | p ≈ 0.0005: 头对显著 |
| 位置基线 | 坐标几何主导 | 正弦 0.60, 线性 0.76: 位置解释大部分 |
| 多项式残差化 | 低阶位置趋势 | 线性残差 0.02, 三次 0.004: 固定对是位置性的 |
| 连续区间 null | 结构域位置混杂 | 富集消失: 非结构域特异 |
| 位置弱头过滤 | 需要位置敏感头 | 双方 pos < 0.10 时 max CKA 0.09 |
| 分段过剩扫描 | 局部 EVO2 优势 | 平均过剩 −0.43 至 −0.48: 无分段优势 |
| 跨突变稳定性 | 突变敏感性 | CKA 0.848 ± 0.009: 对突变不敏感 |
| 对打乱 delta | 突变特异响应 | 观测值 ≈ 对打乱 (p=0.42) |
| Delta 全局模式移除 | 共同扰动轴 | 均值移除后崩塌至 0.04 |
| 局部/接触 delta | 突变位点特异性 | 所有窗口和邻域实际为零 |
| 突变体检索 | 匹配突变体识别 | 随机 (top-1 1%, 平均排名 102) |

---

## 4. 结论

EVO2–RF3 的原始全长 CKA 真实、非随机且头特异，但反映的是深层位置混合诱导的通用低频平滑偏差。在分段和结构域尺度上，显式位置基函数系统性超过真实 EVO2 特征。朴素零假设下表观结构域富集被位置感知控制消除。突变诱导 delta 分析进一步封闭了内容级解释：突变响应空间有序但不具突变特异性。静态和扰动分析均不支持 EVO2 捕获超越坐标几何的结构域特异功能信息。
